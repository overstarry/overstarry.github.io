Augment Context Engine MCP

在 zed 中配置 Augment Context Engine MCP

前言 ACE(Augment Context Engine) 是 Augment 提供的上下文引擎，可以说是 Augment的核心竞争力，12月3号官方正式推出了 ACE MCP,方便在其它 code agent 中使用。 本文将介绍如何在 zed 中配置 ACE MCP。 环境 OS: WSL Ubuntu 24.04 Host OS: Windows 10.0.26100 Zed: v0.216.0+nightly.1979.9857fd233dc22cc2c44fc5c12d8d8620154d0a73 (Zed Nightly) 安装 1 安装 Auggie CLI (Pre-release version) npm install -g @augmentcode/auggie@prerelease 2 登录 augment auggie login 参考 https://docs.augmentcode.com/context-services/mcp/overview
